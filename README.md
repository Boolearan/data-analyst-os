# Data Analyst OS

A personal project to improve my Data Analyst skills while building a practical data ecosystem.

## Objectives

- Improve SQL, Python, BigQuery and Power BI skills.
- Prepare for Senior Data Analyst interviews.
- Track learning progress.
- Organize job applications.
- Analyze personal habits and productivity.
- Practice automation and data workflows.

## Project Modules

### Learning
Track exercises, study sessions, skills, doubts and progress.

### Career
Track job applications, companies, interviews and required skills.

### Personal Analytics
Track study time, habits, productivity and personal progress.

## Tech Stack

- GitHub
- Notion
- Google Sheets
- Python
- BigQuery
- Power BI
